# project-reactjs-github-projects
Project allows you to show your github projects.
Project is written for training purpouse.

Used technology:
- ReactJS
- Redux
- SCSS

To start appliaction(be sure You are on master branch):
```
npm install
npm start
go to http://localhost:7770
```
